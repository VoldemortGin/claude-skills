# 押韵金句优化 Skill

## 适用场景识别

扫描稿件时，重点识别以下句式——它们是押韵改写的最佳候选：

1. **对比/反差句**（A要X，B却Y）
   - 例："人家取的是风和日暖，你非读成抄家问斩"
   - 特征：两个分句形成反差，押韵后喜感倍增

2. **吐槽式总结**（一句话盖棺定论）
   - 例："这flag……真是立得啪啪响"
   - 特征：适合用押韵收束，形成"金句感"

3. **排比递进**（三连击式笑点）
   - 特征：连续短句，节奏一致，末字押韵可产生连珠炮效果

4. **转场金句**（段落之间的承上启下）
   - 特征：一句话概括上文 or 引出下文，押韵让转场更丝滑

5. **回调/呼应句**（前面埋的梗在后面响应）
   - 特征：前后句结构平行，押韵强化呼应感

## 押韵技巧工具箱

### 1. 尾字韵（基础款）
两个分句末字押韵，最自然的口语押韵方式。
- 韵母相同或相近即可，不需要严格平仄
- 优先选择开口音韵母（a, an, ang, ai）——念出来响亮
- 示例：风和日**暖** / 抄家问**斩**（an韵）

### 2. 句式对称（结构款）
两个分句字数相同或接近，内部结构平行。
- 字数差控制在 1-2 字以内
- 动词对动词，名词对名词，但不必死板
- 示例："年号越宏大 / 死得越惨" → 结构平行，节奏对齐

### 3. 反差押韵（杀手锏）
前半严肃、后半荒诞，靠韵脚把两个画风绑在一起，反差本身就是笑点。
- 前半用书面语/正经语气，后半用口语/网络语
- 韵脚越工整，反差越滑稽
- 示例："承天广运圣德神功 / 铜钱刻不下工匠发疯"

### 4. 谐音梗韵（慎用款）
利用谐音制造双关，押韵的同时藏一个梗。
- 只在梗本身有趣时使用，不为谐音而谐音
- 谐音要让人"反应一秒后笑出来"，不能让人皱眉

### 5. 节奏断句韵（进阶款）
不改词，只通过调整断句位置让已有文字产生韵律感。
- 把长句拆成等长短句
- 在关键笑点前"留白"半拍

## 工作流

1. **读取目标**：读取用户指定的文件或段落
2. **扫描候选**：逐段扫描，标记符合上述 5 种场景的候选句子，给出选中理由
3. **生成方案**：对每个候选句提供 2-3 个押韵改写方案，每个方案标注：
   - 改写后的句子
   - 韵脚（标出押韵的字）
   - 押韵类型（尾字韵/句式对称/反差押韵/谐音梗韵/节奏断句韵）
   - 口语顺畅度自评（1-5，5最顺口）
4. **对比展示**：原句 vs 改写方案，方便用户对比选择
5. **用户确认后替换**：用户选择方案后，直接在文件中替换

## 输出格式

对每个候选句，输出如下格式：

```
### 候选 N（第X段）
**原句**：……
**场景类型**：对比/反差句

**方案A**：
> 改写句
- 韵脚：X / Y（an韵）
- 类型：尾字韵
- 顺口度：4/5

**方案B**：
> 改写句
- 韵脚：X / Y（ang韵）
- 类型：反差押韵
- 顺口度：5/5
```

## 约束与红线

1. **事实优先**：不能为了押韵篡改事实、数据、人名、年份。历史科普类稿件尤其注意
2. **口语为王**：播客是念出来的，改写后必须大声念一遍检查——拗口的一律不要
3. **点到为止**：押韵是锦上添花，一篇稿件有 3-5 处金句就够了，不是每句都要押
4. **避免诗词腔**：目标是"脱口秀式顺口"，不是"诗词大会式工整"。保持口语感
5. **不强改已有金句**：如果原文已经很出彩（如"人家取的是风和日暖，你非读成抄家问斩"），标注"原文已是金句，无需修改"，不要画蛇添足
6. **尊重语境**：严肃段落（如李贺的悲剧）不适合硬塞押韵，要识别情绪基调
