# 日韩及东亚（不含中国）史书速查索引

## 日本史书

| 史书 | 年代 | 覆盖范围 | 备注 |
|------|------|---------|------|
| 《古事记》 | 712年 | 神话时代—推古天皇 | 最古史书，神话与史实混杂 |
| 《日本书纪》 | 720年 | 神话时代—持统天皇（697年） | 官修正史，比《古事记》更接近汉籍体例 |
| 《続日本紀》 | 797年 | 文武天皇—桓武天皇（697—791年） | 六国史之二 |
| 《日本三代実録》 | 901年 | 清和—光孝天皇（858—887年） | 六国史最后一部 |
| 《吾妻鏡》 | 约1300年编 | 镰仓幕府时代（1180—1266年） | 幕府官方记录，武家政权史首选 |
| 《愚管抄》 | 慈圆，1220年 | 神武天皇—承久之乱 | 佛教史观编年史 |
| 《太平記》 | 14世纪 | 南北朝时代（1318—1368年） | 军记物语，文学性强，史实需甄别 |
| 《義経記》 | 室町时代 | 源义经生平 | 军记物语，传奇色彩浓 |
| 《信長公記》 | 太田牛一，16世纪末 | 织田信长生平 | 信长时代首要一手史料 |
| 《武功夜話》 | 真伪存疑 | 丰臣秀吉早年 | 史学界对其真实性有争议 |
| 《徳川実紀》 | 19世纪初 | 江户幕府历代将军 | 幕府官修正史 |

## 韩国/朝鲜史书

| 史书 | 作者/年代 | 覆盖范围 | 备注 |
|------|----------|---------|------|
| 《三国史记》 | 金富轼，1145年 | 三国（高句丽、百济、新罗）—统一新罗 | 现存最古朝鲜史书，正史体例 |
| 《三国遗事》 | 一然，1281年 | 三国时代神话、佛教故事 | 神话传说丰富，补充《三国史记》 |
| 《高丽史》 | 郑麟趾等，1451年 | 高丽王朝（918—1392年） | 朝鲜时代官修高丽正史 |
| 《高丽史节要》 | 金宗瑞等，1452年 | 高丽王朝（918—1392年） | 《高丽史》编年体精简版 |
| 《朝鲜王朝实录》 | 各代史官 | 朝鲜王朝（1392—1863年） | 朝鲜王朝最重要官方文献，联合国教科文组织世界记忆遗产 |
| 《承政院日记》 | 各代史官 | 1623—1894年 | 朝鲜王室秘书处日志，细节丰富 |
| 《燃藜室記述》 | 李肯翊，18世纪 | 朝鲜历代掌故 | 笔记体，史料价值高 |

## 越南史书

| 史书 | 作者/年代 | 覆盖范围 | 备注 |
|------|----------|---------|------|
| 《大越史记全书》 | 吴士连等，1479年成书 | 上古（鸿厐氏）—黎朝初期 | 越南最重要正史，以汉文写成 |
| 《钦定越史通鉴纲目》 | 阮朝国史馆，1884年 | 越南历史全期 | 阮朝官修，仿《资治通鉴纲目》体例 |
| 《嘉定城通志》 | 郑怀德，1820年 | 南圻（南越南）地理风俗 | 地方志，南部史地首选 |
